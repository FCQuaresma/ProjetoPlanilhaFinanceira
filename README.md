# Projeto Planilha Financeira

Sistema de controle financeiro desenvolvido para praticar conceitos de desenvolvimento Full Stack, integração entre front-end e back-end, arquitetura em camadas e boas práticas de versionamento.

## Tecnologias utilizadas

### Front-end

- Angular 21
- TypeScript
- Bootstrap
- Angular Material
- Chart.js

### Back-end

- C#
- .NET Web API
- Dapper
- SQL Server

### Ferramentas

- Visual Studio 2022
- Visual Studio Code
- Git
- GitHub

## Funcionalidades

- Cadastro de receitas e despesas
- Edição de movimentações financeiras
- Exclusão de registros
- Filtros por tipo, mês e ano
- Dashboard financeiro
- Indicadores de receitas, despesas e saldo
- Gráficos com Chart.js
- Integração entre Angular e API .NET
- Persistência de dados no SQL Server
